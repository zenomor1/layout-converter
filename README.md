# Layout Converter

## Быстрый старт

### Простая версия (layout_converter.py)
- Два окна, ручной выбор раскладки
- Кнопки Вставить/Конвертировать/Скопировать

### Продвинутая версия (layout_converter_advanced.py)
- ✅ **Автоопределение раскладки** (не нужно выбирать вручную)
- ✅ **Горячая клавиша Shift+Pause** (если доступна)
- ✅ **Быстрая конвертация**: Ctrl+C → Shift+Pause → Ctrl+V

## Использование PRO-версии

1. **Запустите** `LayoutConverterAdvanced.exe`
2. **Сверните** окно (оно работает в фоне)
3. **Выделите текст** с неправильной раскладкой
4. **Нажмите** `Ctrl+C` (копировать)
5. **Нажмите** `Shift+Pause`
6. **Вставьте** `Ctrl+V` — текст будет исправлен!

## Как работает автоопределение

Программа анализирует текст и определяет:
- Если больше русских букв → конвертирует RU→EN
- Если больше английских букв → конвертирует EN→RU

## Требования

Для работы hotkey Shift+Pause нужен модуль `keyboard`. 
Если hotkey не работает, используйте кнопку "Конвертировать сейчас".
